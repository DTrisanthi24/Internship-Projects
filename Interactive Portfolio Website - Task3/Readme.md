~ Interactive Portfolio Webpage
A responsive and interactive personal portfolio built with HTML, CSS, and JavaScript.
This project showcases skills, provides contact functionality, and includes interactive features for a modern user experience.

~ Project Structure
Code
├── index.html   # Main portfolio webpage
├── style.css    # Styling (inside index.html or external if separated)
├── style.js     # JavaScript functionality (form validation + interactivity)
└── README.md    # Project documentation

~ Features
Responsive Design with animated gradient background and glassmorphism cards.

Form Validation

Ensures name, email, and message are filled.

Validates email format.

Displays error/success messages dynamically.

Interactive Features

Dark/Light Mode Toggle

Dynamic Skills Add/Remove

Header Greeting Update

DOM Manipulation for updating content dynamically.

Reusable Functions ($, showMessage, validateForm, toggleTheme, addSkill, updateGreeting).

Event Listeners for form submission, button clicks, and header interaction.

~ Technologies Used
HTML5 for structure

CSS3 for styling (gradients, animations, responsiveness)

JavaScript (ES6) for interactivity and validation

~ How to Use
Clone the repository:

bash
git clone https://github.com/yourusername/portfolio.git
Open index.html in your browser.

Interact with the portfolio:

Submit the contact form (validation included).

Toggle between dark/light mode.

Add/remove skills dynamically.

Click the header title to update the greeting.
